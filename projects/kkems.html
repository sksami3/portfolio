ExpenseTracker

  • Tech Stack: .NET 6, Angular, Entity Framework, MS SQL
  • Github URL: Project Link

Project Scope: ExpenseTracker is a comprehensive expense tracking software designed to help individuals manage and monitor their financial transactions with family and friends.

System Functionality:

  • Create and manage user accounts with relation settings.
  • Organize expenses into relation groups for collaborative tracking.
  • Generate monthly expense reports for comprehensive financial analysis.
  • View real-time expense states and updates.
  • Categorize expenses and apply tags for efficient tracking and analysis.
  • Enable sharing and splitting of expenses among family and friends.
